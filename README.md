# RL Game

RL (Real Game) is a polygon-based game written in C++ using Vulkan for graphics rendering. The game features a voxel-based world that players can explore and interact with. The game includes a variety of biomes, creatures, and resources to discover, and allows players to explore the space and build spacecrafts to explore the space and other planets. The coordinate system is based on a 3D grid, where each 1000 Units represents a unit in the grid (every 0.001 Unit = 1 Meter, represented using the float data type)

## Features
- Polygon-based world (or similar)
- Space exploration
- Spacecraft building
- Multiplayer support
- Cross-platform support (Windows, Linux, macOS)
- Lightweight and easy to run on a wide range of devices
- Written in C++ using Vulkan for graphics rendering (high performance)
- Open source and free to use
